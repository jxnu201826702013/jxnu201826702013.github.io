Markdown入门指南

博客搭建教程可以参考过客~励心的博客，感谢这位大佬，我的博客就是在这篇文章的指导下搭好的。大佬技术好，人也很好，愿意帮助萌新，开始搭博客只能还特意去学了下Markdown语法（偷看，十分钟就能学会），因为hexo+GitHub部署博客写文章用的是Markdown格式。另外Markdown是兼容HTML的，这里介绍一下Markdown常用的几种语法。(写Markdown推荐Typora)

  • 标题

    一个#代表一级标题，两个#代表二级标题，最高六级，#和标题之间一个空格

    # 一级标题

    # 二级标题

    # 三级标题

    一级标题

    二级标题

    三级标题

  • 字体加粗

    加粗的文字左右两边分别用两个* 包起来

    **我是加粗的文字**

    我是加粗的文字

  • 字体倾斜

    斜的文字左右分别用一个* 号包起来

    *我是倾斜的文字*

    我是倾斜的文字

  • 文字斜体加粗

    和加粗的文字左右分别用三个* 号包起来

    ***我是既倾斜又加粗的文字***

    我是既倾斜又加粗的文字

  • 删除线

    删除线的文字左右分别用两个~ 号包起来

    ~~我是加删除线的文字~~

    我是加删除线的文字

  • 引用

    用的文字前加一个>，引用可以嵌套，如两个>>

    >我是引用的文字

    我是引用的文字

  • 分割线

    三个的-或者*

  • 图片

    格式：
    ![图片](https://raw.githubusercontent.com/8546KB/My_picture/master/20191207192815.png)

  • 超链接

    格式：
    [百度](https://www.baidu.com/)

    百度

  • 无序列表

    无序列表用 - + * 任何一种都可以

    +列表内容
    - 列表内容
    * 列表内容

    • 列表内容
    • 列表内容
    • 列表内容

  • 有序列表
    数字加点

    1.列表内容
    2.列表内容
    3.列表内容

    1.列表内容
    2.列表内容
    3.列表内容

  • 单行代码

    代码之间分别用一个反引号包起来

    `System.out.print(“hello world);'

    Sysem.out.print("hello world");

  • 多行代码

    代码之间分别用三个反引号包起来，且两边的反引号单独占一行,加上语言

    ```c
    #include<stdio.h>
    int main()
    {
    printf(“hello world);
    return 0;
    }
    ```

    #include<stdio.h>
int main() 
{
​    printf("hello world);
​    return 0;
}

  • 表格

    格式：
    姓名|技能|排行
    –|:–:|–:
    刘备|哭|大哥
    关羽|打|二哥
    张飞|骂|三弟

姓名 技能 排行
刘备 哭 大哥
关羽 打 二哥
张飞 骂 三弟
